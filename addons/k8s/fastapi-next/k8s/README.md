# Kubernetes (optional add-on) — {{ project_name }}

> **Optional and not free-tier.** A cluster + managed Postgres are not free.
> This scaffold is here when you outgrow the one-button Cloudflare deploy
> (`make deploy`). It is **not** generated by default — re-run firestarter with
> `--set include_k8s=yes` (or answer `yes` at the prompt) to include it.

Kustomize manifests for deploying the FastAPI backend + Next.js frontend.

```
k8s/
├── base/                 # shared Deployments/Services + namespace
└── overlays/
    ├── staging/          # 1 replica, staging namespace
    └── production/       # 2 replicas, ingress, prod namespace
```

## Apply

```bash
kubectl apply -k k8s/overlays/staging      # or .../production
```

## Database & secrets — bring your own (managed)

This scaffold intentionally does **not** run Postgres/Redis in-cluster — use a
managed service (RDS, Cloud SQL, Neon, Upstash, …) and provide the connection
strings via a Secret. `base/secret.example.yaml` shows the shape; create the
real one out-of-band (sealed-secrets / external-secrets / `kubectl create secret`).
**Never commit real secret values.**

## Images

Build and push your own; the manifests reference `{{ project_slug }}-backend:latest`
and `{{ project_slug }}-frontend:latest`. Set your registry in the overlay
`images:` field.
